---
name: tradingview-research-discipline
description: Use TradingView safely as a chart/research surface without confusing chart data with executable broker data. Use for TradingView symbols, chart links, watchlists, indicators, screenshots, or chart-based research.
---

Use `market_tradingview_chart_url` for a public chart link. Record exchange/provider-qualified symbol, resolution, timezone, chart retrieval time, and whether feed is delayed/composite/mid/bid/ask.

TradingView chart availability is not an API entitlement or execution feed. Do not scrape authenticated pages, automate login, place orders, or infer executable fills from a chart.
