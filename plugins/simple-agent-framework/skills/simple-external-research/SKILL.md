---
name: simple-external-research
description: Use Perplexity and Firecrawl MCP tools safely for MatinDeevv/simple research, documentation, and source verification. Use when live web evidence, deep research, scraping, crawling, or primary-source extraction is needed.
---

Perplexity tools need `PERPLEXITY_API_KEY`; Firecrawl tools need `FIRECRAWL_API_KEY`. Keys stay in environment only. Never place keys in files, prompts, receipts, commits, or chat.

Use Perplexity for discovery/synthesis. Use Firecrawl only for allowed public URLs and least-cost operation: search/scrape before batch/crawl. Respect site terms, robots, limits, and citations. Store only source metadata and content hashes unless user explicitly requests retained content.

Cross-check material claims with primary sources. Do not use external research as profitability, causal, or promotion proof.
