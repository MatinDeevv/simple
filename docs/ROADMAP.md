# Roadmap

This roadmap describes the planned evolution of Azar. Dates are targets, not commitments.

## Q3 2026

- [ ] Stabilize classical integrator on synthetic burn-in data.
- [ ] Finalize identity-aware coupling diagnostics and reporting.
- [ ] Complete documentation site on GitHub Pages.
- [ ] Add benchmark harness for classical vs. naive comparator.

## Q4 2026

- [ ] Ingest first post-2024 holdout split (predeclared).
- [ ] Run OQ-14 gate evaluation on residual-level research.
- [ ] Publish first evaluation run manifest with evidence artifacts.
- [ ] Add containerized reproducibility environment.

## 2027

- [ ] Expand instrument coverage beyond the initial ten FX pairs.
- [ ] Investigate regime-change detection for coupling re-estimation.
- [ ] Open selective quantum archive experiments to community reproduction.
- [ ] Release stable v1.0 API for classical dynamics.

## Long-term research questions

- Can causal coupling be estimated reliably at sub-hourly horizons?
- What is the minimal feature set needed for stable integrator replay?
- Do quantum kernels provide any measurable advantage over classical baselines on FX residuals? (Current archive: no evidence found.)
